# 2026-autoware-offroad-sim
Replicating simulation environment for Autoware off-road racing

Use [`justfile`](justfile) to install and launch Isaac Sim 6.0.

The [Nova Carter example](https://docs.isaacsim.omniverse.nvidia.com/6.0.0/assets/nova_carter_landing_page.html) is a great way to get started. ([screenshots](https://drive.google.com/drive/folders/116wFktTVzmDK4k1pCN9Z_srilShDyCUa?usp=sharing))
